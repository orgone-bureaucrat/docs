## What is Kisuke?

Kisuke is a mobile coding app that brings a full development environment to your phone or tablet. With built-in AI chat, terminal, and code editor, you can code from anywhere.

    ```bash
    # Install via CLI
    curl -fsSL https://kisuke.app/install.sh | sh

    # Or use Docker
    docker pull kisuke/kisuke-connect
    ```

  </Tab>

## Sign In Options

Kisuke supports multiple authentication providers:

- **Claude** - Sign in with your Anthropic account
- **Google** - Sign in with Google
- **OpenAI** - Sign in with your OpenAI account
