## Overview

Connect your phone to remote servers for:

- Managing cloud VMs
- Deploying applications
- Accessing development servers
- Running commands on remote machines

## Prerequisites

<Steps>
  <Step title="Install Kisuke on your phone">
    Download from [App Store](https://apps.apple.com/app/kisuke) or [Google Play](https://play.google.com/store/apps/details?id=app.kisuke)
  </Step>
  <Step title="Install Kisuke Connect on your server">
    Use [CLI or Docker](/installation/cli-docker) for headless installation
  </Step>
  <Step title="Generate an auth key">
    Create an auth key in Kisuke app settings
  </Step>
</Steps>

## Server Setup

### Install Kisuke Connect

```bash
# Quick install
curl -fsSL https://kisuke.app/install.sh | sh
```

### Authenticate with Auth Key

Since servers typically don't have browsers, use an auth key:

```bash
kisuke-connect login --auth-key=tskey-xxxxxxxx
```

<Warning>
Generate auth keys from your Kisuke app under **Settings** > **Auth Keys**. Keep them secure.
</Warning>

### Start the Service

```bash
# Start as daemon
kisuke-connect up --daemon

# Or use systemd
sudo systemctl enable --now kisuke-connectd
```

## Docker Setup

For containerized deployments:

```bash
docker run -d \
  --name kisuke-connect \
  --restart unless-stopped \
  -e KISUKE_AUTH_KEY=tskey-xxxxxxxx \
  -e KISUKE_HOSTNAME=my-server \
  kisuke/kisuke-connect
```

## Connecting from Your Phone

1. Open Kisuke on your phone
2. Go to **Connect** > **Devices**
3. Your server should appear in the list
4. Tap to connect

## Use Cases

### Cloud VMs

Connect to AWS EC2, Google Cloud, Azure, DigitalOcean, or any cloud VM:

```bash
# On your cloud VM
curl -fsSL https://kisuke.app/install.sh | sh
kisuke-connect login --auth-key=your-key
kisuke-connect up --daemon
```

### Development Servers

Access your dev server from anywhere:

- Run builds remotely
- Tail logs
- Deploy applications

### Home Lab

Connect to your home server when away:

- Access media servers
- Manage home automation
- Check on running services

## Security Considerations

<Warning>
Server connections provide terminal access to your machine. Ensure you:

- Use strong authentication on the server itself
- Keep Kisuke Connect updated
- Rotate auth keys regularly
- Monitor connected devices
</Warning>
