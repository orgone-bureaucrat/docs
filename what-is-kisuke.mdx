## Overview

Kisuke is a mobile-first coding application that brings a complete development environment to your iOS or Android device. Whether you're on the go or away from your desk, Kisuke lets you write code, run commands, and chat with AI assistants.

## Key Features

### AI Chat

Kisuke includes a built-in AI chat interface that supports multiple providers:

- **Claude** (Anthropic)
- **GPT** (OpenAI)
- **Google AI**

Sign in with your existing account to start coding with AI assistance directly from your phone.

### Terminal & Shell

Access a full terminal environment on your mobile device. Run commands, execute scripts, and manage your development workflow.

### Code Editor

A mobile-optimized code editor with:

- Syntax highlighting
- File management
- Project navigation

### Kisuke Connect

Connect your mobile device to your desktop or server using Kisuke Connect - our secure networking solution (similar to Tailscale). This allows you to:

- Access your development machine from anywhere
- Run commands on remote servers
- Sync files between devices

## How It Works

1. **Install Kisuke** on your iOS or Android device
2. **Sign in** with Claude, Google, or OpenAI
3. **Connect** to your PC or server using Kisuke Connect
4. **Start coding** with terminal, editor, and AI chat

## Use Cases

- **Mobile development** - Code while commuting or traveling
- **Server management** - SSH into servers from your phone
- **Quick fixes** - Make hotfixes without needing your laptop
- **Learning** - Practice coding with AI assistance anywhere
